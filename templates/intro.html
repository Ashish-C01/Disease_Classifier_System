About the web application

The project entitled Disease classification system is web application through which a patient can check if he/she has any of the diseases described below or not.

This system can detect following diseases:

  • Pneumonia
  • Liver disease
  • Malaria
  • Heart attack

The main objective of this web application is to reduce the amount of time spent while waiting for the report.

A right system can save a lot of money and also the time wasted while waiting for the report regarding the corresponding test.

Heart attack detection

  • In this module the user can check if he/she has chances of getting heart attack or not
  • k nearest neighbor algorithm has been used to train the model
  • K-Nearest Neighbour is a Machine Learning algorithm based on Supervised Learning technique. K-NN algorithm assumes the similarity between the new case/data and available cases and put the new case into the category that is most similar to the available categories.
  • The model has 84.7% accuracy on training data and 91.80% accuracy on test data
  • The recall of this model for detection is good

Liver disease detection

  • In this module the user can check if he/she has chances of getting liver disease or not
  • Deep learning has been used to train the model
  • The model has 89.92% accuracy on training data and 86.23% accuracy on test data
  • The user will have to provide numerical values as an input
